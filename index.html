מה זה?

דשבורד אישי למעקב לימודים בקורס סיעוד המבוגר. המערכת עוקבת אחר ההתקדמות שלך ב-18 נושאי לימוד, מאורגנים בחלק א׳ וחלק ב׳.

איך פותחים

  • מכל מכשיר, כולל טלפון: rafimichaeli.github.io/nursing-study-tracker
  • מקומית: פתיחת הקובץ ישירות בדפדפן, ללא צורך בהתקנה.

התמצאות בממשק

menu סרגל ניווט שמאלי — מסך רחב

  • מציג את אחוז ההתקדמות הכולל ופירוט לפי חלק א׳ וחלק ב׳.
  • לחיצה על "חלק א׳" או "חלק ב׳" מסננת את הטבלה. לחיצה שנייה מבטלת.
  • קישורים מהירים לקטגוריות: כירורגיה, לב, נשימה וכו׳.
  • בתחתית הסרגל: כפתורי שמור / טען / אפס הכל / הוראות שימוש.

smartphone ניווט במובייל

  • סמל ה-☰ פותח את סרגל הניווט המלא כתפריט נגלל, כולל כפתורי השמירה וההוראות בתחתיתו.
  • סרגל תחתון קבוע עם 5 כפתורים: הכל, חלק א׳, חלק ב׳, נותר, ומראה — לגישה מהירה.

view_headline סרגל עליון

  • מונה עמודים: כמה עמודי ברונר הושלמו מתוך הסך הכולל.
  • אחוז התקדמות כולל עם פס התקדמות.

כל שורה מייצגת נושא לימוד ומציגה: שם, קטגוריה, עמודי ברונר, שעות, סטטוס ואחוז התקדמות.

שמירה וטעינה

info ההתקדמות נשמרת אוטומטית ב-localStorage, אך מומלץ לשמור גיבוי חיצוני.

save שמירה ידנית

לחץ "שמור התקדמות". בפעם הראשונה תבחר מיקום לקובץ progress.json. בפעמים הבאות נשמר אוטומטית לאותו מיקום.

autorenew שמירה אוטומטית

לאחר שמירה ידנית ראשונה — המערכת שומרת כל 5 דקות. ה-chip בסרגל יהפוך לירוק ויציג את שעת השמירה האחרונה.

info שמירה/טעינה אוטומטית לקובץ עובדת רק ב-Chrome / Edge. ב-Safari וב-Firefox (כולל אייפון) כל שמירה תוריד קובץ ידנית — ההתקדמות עצמה עדיין נשמרת אוטומטית בדפדפן בכל מקרה.

folder_open טעינה

לחץ "טען התקדמות" לטעינת קובץ שנשמר קודם — שימושי בעת מעבר למחשב אחר.

delete איפוס

"אפס הכל" מוחק את כל הסימונים בדפדפן הזה. פעולה בלתי הפיכה — שמור גיבוי לפני שימוש.
